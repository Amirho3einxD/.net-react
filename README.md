# Event Management System

سامانه مدیریت رویدادها یک پروژه Full Stack است که با استفاده از ASP.NET Core Web API در بخش Backend و React + TypeScript در بخش Frontend توسعه داده شده است. این پروژه با هدف پیاده‌سازی معماری‌های مدرن توسعه نرم‌افزار، مدیریت کاربران، ارتباط بلادرنگ و ارائه تجربه کاربری مناسب طراحی شده است.

کاربران می‌توانند پس از ثبت‌نام و ورود به سیستم، رویدادها را مشاهده و مدیریت کرده، در رویدادها شرکت کنند، با سایر کاربران تعامل داشته باشند و از قابلیت‌های مختلف سامانه استفاده کنند.

## امکانات پروژه

* ثبت‌نام و ورود کاربران
* احراز هویت و مجوزدهی مبتنی بر Identity
* مدیریت رویدادها و فعالیت‌ها
* ایجاد، ویرایش و حذف رویدادها
* شرکت در رویدادها
* مشاهده اطلاعات کاربران و پروفایل‌ها
* ارتباط بلادرنگ با SignalR
* نمایش موقعیت‌ها روی نقشه
* فرم‌های پیشرفته همراه با اعتبارسنجی
* مدیریت وضعیت برنامه با MobX
* مدیریت داده‌ها و درخواست‌های سرور با React Query
* مدیریت خطاها و اعلان‌های کاربری
* معماری چندلایه و قابل توسعه

## تکنولوژی‌های استفاده شده

* Axios

## هدف پروژه

هدف از توسعه این پروژه یادگیری و پیاده‌سازی مفاهیم پیشرفته توسعه وب از جمله معماری چندلایه، مدیریت وضعیت در برنامه‌های React، ارتباط Frontend و Backend، احراز هویت کاربران، ارتباط بلادرنگ و توسعه برنامه‌های Full Stack با تکنولوژی‌های روز بوده است.
