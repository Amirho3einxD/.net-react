# پروژه Full Stack با ASP.NET Core و React

این پروژه یک وب‌اپلیکیشن Full Stack است که با استفاده از ASP.NET Core Web API در بخش Backend و React در بخش Frontend توسعه داده شده است. در طراحی این پروژه از معماری چندلایه و الگوهای مدرن توسعه نرم‌افزار استفاده شده تا کدها از نظر نگهداری، توسعه‌پذیری و خوانایی در سطح مناسبی قرار بگیرند.

در بخش سرور از Entity Framework Core برای مدیریت داده‌ها، ASP.NET Identity برای احراز هویت کاربران، MediatR برای پیاده‌سازی الگوی CQRS و FluentValidation برای اعتبارسنجی اطلاعات استفاده شده است. همچنین قابلیت ارتباط لحظه‌ای (Real-Time) با استفاده از SignalR در پروژه پیاده‌سازی شده است.

## ویژگی‌های پروژه

* توسعه Frontend با React
* توسعه Backend با ASP.NET Core Web API
* معماری چندلایه (Layered Architecture)
* استفاده از Entity Framework Core
* احراز هویت و مدیریت کاربران با ASP.NET Identity
* پیاده‌سازی CQRS با MediatR
* اعتبارسنجی داده‌ها با FluentValidation
* ارتباط Real-Time با SignalR
* مدیریت خطاها با Middleware اختصاصی
* استفاده از AutoMapper برای تبدیل مدل‌ها
* پایگاه داده SQLite

## تکنولوژی‌های استفاده شده

### Backend

## هدف پروژه

هدف از توسعه این پروژه یادگیری و پیاده‌سازی مفاهیم پیشرفته توسعه وب از جمله معماری چندلایه، مدیریت کاربران، ارتباط بین Frontend و Backend، ارتباط لحظه‌ای و استفاده از الگوهای رایج در توسعه نرم‌افزار بوده است.
